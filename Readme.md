# GameConsole

> ## 特性

- 基于Unity的日志模块。
- 在Gameplay中模拟控制台程序执行命令，可绑定实例/静态方法，且支持多个参数。
- 支持缓存执行过的命令，支持命令自动补全。
- 使用UGUI作为界面，对ScrollView进行了优化，支持调节Console大小。

> ## 快速开始

1. 拷贝目录GameConsole到指定目录，并导入TMP_Pro。
2. 将Prefab ：GameConsole，拖入到场景中即可，另外得添加一个EventSystem。

    > 项目版本为2019.3b，可以直接参考示例场景
